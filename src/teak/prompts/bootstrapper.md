# Brain Bootstrapper Prompt

You are bootstrapping the Project Brain for a codebase you are about to work with.
Read the codebase survey provided and produce four Markdown files:

- ARCHITECTURE.md — major components, data flow, system design
- CONVENTIONS.md — naming, patterns, what to avoid
- DECISIONS.md — ADRs explaining why things are the way they are
- MEMORY.md — open questions, cross-session notes, current focus

Rules:
- Be concise. The user will review every line.
- Mark any guess with `(?)` so the user can correct it during approval.
- Do not invent decisions you cannot find evidence for.
